Printable Scenario Creator V1

Random single-line pump practice sheets.

Generate four beginner driver/operator problems per page using a clean black-and-white scene, blank student work space, and an optional answer key.

Build worksheet

V1 keeps it simple: single attack line problems only. Later versions can add wye, standpipe, master stream, and department-selected equipment.

The math uses PP = NP + FL ± Elevation + Appliance Loss and FL = C × (GPM/100)² × hose length in hundreds of feet.
