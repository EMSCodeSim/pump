How to use the table

Pick the hose length closest to the evolution, then find the desired flow. The number shown is estimated friction loss for that hose size and length. Add nozzle pressure, appliance loss, and elevation to estimate pump discharge pressure.

Example: 1¾″ C15.5185 GPM200 ftFL ≈ 105 psiPDP with 50 psi nozzle ≈ 155 psi
